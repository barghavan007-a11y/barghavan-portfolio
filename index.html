Still Life

A cinematic still life project focused on composition, shading, and visual mood. This work explores realistic materials, lighting balance, and polished presentation through procedural workflows. The project emphasizes a refined visual style while maintaining strong form, texture, and cinematic framing.

Turtle Quilt

Turtle Quilt is a Python-based generative design project developed through coded pattern creation. It explores repetition, structure, color rhythm, and algorithmic composition to build a quilt-inspired visual. The focus of the project is on procedural logic and creative image generation through Python workflows.

Data Visualization

Data Visualization is a Python-driven project that transforms structured data into a stylized visual output. The work focuses on clarity, composition, and readable design while presenting information in a visually engaging way. It combines analytical structure with a cinematic presentation approach.
